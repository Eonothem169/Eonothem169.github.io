github pages，自定义域名开启https遇到Domain does not resolve to the GitHub Pages server

背景：

利用Hexo搭建博客，想用自定义域名。结果在github page页面遇到DNS解析不成功。

 

解决方法：

1.在域名工作台删除所有主机记录.

2.打开CMD，输入命令ping “yourname”.github,io 获取到IP地址.

3.在域名工作台里添加两个记录，如图（把记录值换成上面获取到的IP地址).

4.在Blog/source里添加一个名字为CNAME的文件，里面输入你的域名（注意不要加www).

5.在page页面的custom domain中输入你的域名，进入你的域名，刷新，发现成功显示.

改进之处：

如最后一张图片中显示，目前想到的改进措施：把主机记录中的www那一条删除。不过现在我懒得弄了。因为要准备期末……
